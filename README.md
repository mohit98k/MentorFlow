Its my first full-stack project . the purpose of this project are---->  
    
    1.To give you an optimal roadmap for a particular skill you want to learn along with the required resouce list
    2.help you analyze your resume, ats score calculate and fix whatever is wrong in it 
    3.track the status of the  jobs you've applied for 
    4.maintain the list of skills you have acquired

Frontend------>

    React (Vite)

    JavaScript

    Tailwind CSS

    Axios

    React Router


Backend------>

    Node.js

    Express.js

    MongoDB Atlas

    Mongoose

    JWT Authentication

    Google Gemini API (Free Tier)

    Cloudinary (File Handling)


Deployment------>

    Frontend: Vercel

    Backend: Render

    Database: MongoDB Atlas


Authentication------>

    JWT-based authentication

    Secure password hashing (bcrypt)

    Protected routes

    Token-based authorization

    CORS-restricted backend access
